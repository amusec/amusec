# Vivek Kumar | Still Curious

A cinematic personal website that tells the story of curiosity, experiments, servers, security, failures, and the search for real challenges.

This is not a traditional portfolio or resume page. It is designed as a slide-based storytelling experience, where each full-screen section represents one chapter of the journey.

## About The Story

The website follows my path from discovering HTML in a Class 7 computer textbook to building websites, exploring domains, hosting servers from home, learning networking during Covid, working on a Webflow exporter project, teaching C, graduating in Computer Science, and finding real-world work through security testing and product projects.

The core idea is simple:

> I am not looking for a job. I am looking for the next problem that keeps me awake at 3 AM.
